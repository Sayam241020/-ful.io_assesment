Q1. What is reverse-i-search in a Linux terminal?

Ans:- reverse-i-search (also known as reverse incremental search) is a feature that allows you to search through your command history for a previously entered command by typing a portion of it.

ctrl+R is used to initiate the process. we can keep pressing until we cannot get our match and then execute it.



Q2. How do you list out the hidden directories using Linux command line tool?

Ans:- ls -a | grep '^\.' can be used to list out all hidden directories. ls -a shows all hidden directories and grep command is used to filter out 
hidden directories
